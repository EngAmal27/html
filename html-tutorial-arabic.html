ما هو HTML؟
الدرس الأول — مقدمة وأساسيات
مبتدئ

HTML اختصار لـ HyperText Markup Language، وهي الـ لغة الأساسية اللي بيتبنى بيها الويب كله. مش لغة برمجة بالمعنى الحرفي، لكنها لغة توصيف بتخليك تقول للمتصفح إزاي يعرض المحتوى.

فكر فيها كأنها هيكل العضام للصفحة — CSS بتاخد دور المظهر والجمال، وJavaScript بتاخد دور الحركة والتفاعل. لكن من غير HTML، مفيش حاجة تتبني عليها!

💡
معلومة سريعة HTML اتعملت سنة 1991 على إيد Tim Berners-Lee، واللي بيعتبر أبو الإنترنت. النسخة الحالية هي HTML5 اللي اتنزلت رسمياً سنة 2014.
HTML
<!-- أول HTML في حياتك 🎉 --> <!DOCTYPE html> <html> <head> <title>أول صفحة ويب</title> </head> <body> <h1>أهلاً بالعالم!</h1> <p>ده أول كود HTML بكتبه 🚀</p> </body> </html>
🏗️
هيكل صفحة HTML
الدرس الثاني — الهيكل الأساسي
مبتدئ

كل صفحة HTML لازم يكون ليها هيكل ثابت. زي ما البيت عنده أساس وحوائط وسقف، الصفحة عندها <!DOCTYPE> وـ <head> وـ <body>.

الوسم وظيفته
<!DOCTYPE html>بيخبر المتصفح إن الصفحة دي HTML5
<html>الجذر الأساسي للصفحة كلها
<head>معلومات الصفحة (مش بتتعرض للزوار)
<title>اسم الصفحة اللي بيظهر في التاب
<meta>معلومات إضافية زي الترميز واللغة
<body>كل المحتوى المرئي بتاع الصفحة
⚠️
لازم تعرفها دايماً حط charset="UTF-8" في الـ meta عشان الحروف العربية تتعرض صح!
HTML — الهيكل الكامل
<!DOCTYPE html> <html lang="ar" dir="rtl"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>عنوان الصفحة</title> </head> <body> <!-- المحتوى هنا --> </body> </html>

العناوين — Headings
الدرس الثالث — من h1 لـ h6
مبتدئ

HTML عندها 6 مستويات للعناوين من h1 (الأكبر والأهم) لـ h6 (الأصغر). المتصفحات والـ SEO بيولوا أهمية كبيرة للـ h1 — خليه واحد بس في الصفحة.

HTML
<h1>عنوان رئيسي - الأكبر</h1> <h2>عنوان ثانوي</h2> <h3>عنوان من المستوى الثالث</h3> <h4>عنوان من المستوى الرابع</h4> <h5>عنوان من المستوى الخامس</h5> <h6>عنوان صغير - المستوى السادس</h6>
🧠 اختبر نفسك
إيه العنوان الأهم لمحركات البحث؟
أ <h6>
ب <h1>
ج <title>
د <header>

النصوص والتنسيق
الدرس الرابع — p, strong, em, br, hr
مبتدئ

النصوص في HTML بتتكتب جوه وسوم مخصصة لكل نوع. أهمهم عنصر الفقرة <p> واللي بيكون المحتوى النصي الأساسي.

<p>

فقرة نصية عادية

<strong>

نص عريض Bold مهم

<em>

نص مائل Italic

<br>

سطر جديد (بدون إغلاق)

<hr>

خط أفقي فاصل

<span>

تنسيق جزء من النص

<mark>

تظليل نص (HTML5)

<small>

نص أصغر حجماً

HTML
<p>ده نص عادي في فقرة.</p> <p>ده نص فيه <strong>جزء مهم</strong> وجزء <em>مائل</em>.</p> <p> السطر ده<br> والسطر ده في نفس الفقرة. </p> <hr> <p>نص تحت الخط الفاصل.</p>

الروابط — Links
الدرس الخامس — وسم <a>
مبتدئ

الروابط هي روح الويب! كل رابط بتشوفه على الإنترنت هو عنصر <a> بسيط. الخاصية الأساسية هي href اللي بتحدد وجهة الرابط.

HTML
<!-- رابط خارجي --> <a href="https://google.com" target="_blank"> افتح Google في تاب جديد </a> <!-- رابط داخلي في نفس الموقع --> <a href="about.html">صفحة عننا</a> <!-- رابط لجزء في نفس الصفحة --> <a href="#contact">اتصل بينا</a> <!-- رابط إيميل --> <a href="mailto:test@example.com">ابعتلنا ميل</a>
✅
نصيحة استخدم target="_blank" للروابط الخارجية عشان تفتح في تاب جديد وتحافظ على المستخدم في موقعك.
🖼️
الصور — Images
الدرس السادس — وسم <img>
مبتدئ

وسم <img> بيعرض الصور في الصفحة. لاحظ إنه وسم ذاتي الإغلاق (Self-closing) يعني مفيش </img>.

HTML
<!-- صورة أساسية --> <img src="photo.jpg" alt="وصف الصورة"> <!-- صورة بعرض وارتفاع محددين --> <img src="logo.png" alt="الشعار" width="200" height="100"> <!-- صورة من الإنترنت --> <img src="https://example.com/img.jpg" alt="صورة من الإنترنت">
🚨
ضروري جداً! دايماً حط خاصية alt — دي مهمة لمحركات البحث وللمكفوفين اللي بيستخدموا قارئات الشاشة. من غيرها موقعك مش accessible!
📋
القوائم — Lists
الدرس السابع — ul, ol, dl
مبتدئ

في HTML أنواع تلاتة من القوائم. كل نوع ليه استخدام مختلف حسب المحتوى.

HTML
<!-- قائمة غير مرتبة (نقاط) --> <ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul> <!-- قائمة مرتبة (أرقام) --> <ol> <li>الخطوة الأولى</li> <li>الخطوة الثانية</li> <li>الخطوة الثالثة</li> </ol> <!-- قائمة تعريفات --> <dl> <dt>HTML</dt> <dd>لغة توصيف الويب</dd> </dl>

الجداول — Tables
الدرس الثامن — table, tr, th, td
متوسط

الجداول بتُستخدم لعرض البيانات في صفوف وأعمدة. كل جدول بيتكون من rows (صفوف) وكل صف بيتكون من cells (خانات).

HTML
<table border="1"> <!-- رأس الجدول --> <thead> <tr> <th>الاسم</th> <th>العمر</th> <th>المدينة</th> </tr> </thead> <!-- جسم الجدول --> <tbody> <tr> <td>أحمد</td> <td>25</td> <td>القاهرة</td> </tr> </tbody> </table>

النماذج — Forms
الدرس التاسع — form, input, button
متوسط

النماذج هي الطريقة الأساسية لاستقبال بيانات من المستخدم — تسجيل دخول، بحث، إرسال بيانات، إلخ. كل حاجة بتكتبها على الإنترنت هي Form!

HTML
<form action="/submit" method="post"> <!-- حقل نص --> <label for="name">الاسم:</label> <input type="text" id="name" placeholder="اكتب اسمك"> <!-- حقل إيميل --> <input type="email" placeholder="البريد الإلكتروني"> <!-- كلمة مرور --> <input type="password" placeholder="كلمة المرور"> <!-- زر الإرسال --> <button type="submit">إرسال</button> </form>

العناصر الدلالية — Semantic HTML
الدرس العاشر — header, nav, main, footer
HTML5

من أهم مميزات HTML5 هي العناصر الدلالية (Semantic Elements). قبل HTML5 كنا بنستخدم <div> لكل حاجة — دلوقتي عندنا عناصر ليها معنى واضح.

<header>

رأس الصفحة أو القسم

<nav>

قائمة التنقل

<main>

المحتوى الرئيسي

<section>

قسم من المحتوى

<article>

مقالة أو محتوى مستقل

<aside>

محتوى جانبي

<footer>

تذييل الصفحة

<figure>

صورة مع تعليق

HTML5 — هيكل صفحة احترافي
<body> <header> <nav>... قائمة التنقل ...</nav> </header> <main> <section> <article> <h2>عنوان المقالة</h2> <p>محتوى المقالة...</p> </article> </section> <aside>... محتوى جانبي ...</aside> </main> <footer> <p>جميع الحقوق محفوظة © 2025</p> </footer> </body>

الميديا — Audio & Video
الدرس الحادي عشر — video, audio
HTML5

HTML5 جاب دعم native للفيديو والصوت من غير ما تحتاج Flash أو أي plugin خارجي. دلوقتي أي متصفح حديث بيشغل الميديا مباشرة!

HTML5
<!-- فيديو --> <video width="640" height="360" controls> <source src="movie.mp4" type="video/mp4"> <source src="movie.ogg" type="video/ogg"> متصفحك مش بيدعم الفيديو. </video> <!-- صوت --> <audio controls> <source src="song.mp3" type="audio/mpeg"> متصفحك مش بيدعم الصوت. </audio> <!-- فيديو يوتيوب عبر iframe --> <iframe src="https://www.youtube.com/embed/VIDEO_ID" width="560" height="315" allowfullscreen></iframe>

Canvas و SVG
الدرس الثاني عشر — رسم في المتصفح
HTML5

Canvas بيخليك ترسم جرافيك وألعاب وأشكال بـ JavaScript مباشرة. SVG لغة رسوميات قابلة للتكبير — مثالية للأيقونات والشعارات.

HTML5 Canvas
<!-- Canvas element --> <canvas id="myCanvas" width="400" height="200"></canvas> <script> const canvas = document.getElementById('myCanvas'); const ctx = canvas.getContext('2d'); // ارسم مستطيل أحمر ctx.fillStyle = '#ef4444'; ctx.fillRect(10, 10, 150, 100); // ارسم نص ctx.fillStyle = 'white'; ctx.font = '20px Cairo'; ctx.fillText('مرحباً!', 40, 65); </script>
💾
Local Storage
الدرس الثالث عشر — تخزين بيانات في المتصفح
HTML5

HTML5 جاب Web Storage API اللي بتخليك تحفظ بيانات في المتصفح من غير ما تحتاج Cookies. في نوعين: localStorage (بيفضل حتى بعد إغلاق المتصفح) وsessionStorage (بيتمسح لما تغلق التاب).

JavaScript + HTML5
// حفظ بيانات localStorage.setItem('username', 'أحمد'); localStorage.setItem('theme', 'dark'); // استرجاع بيانات const user = localStorage.getItem('username'); console.log(user); // أحمد // حذف بيانات localStorage.removeItem('username'); // مسح كل البيانات localStorage.clear();

Forms الجديدة في HTML5
الدرس الرابع عشر — أنواع input جديدة
HTML5

HTML5 أضاف أكتر من 10 أنواع جديدة لعنصر input — بيوفروا validation تلقائي وواجهات مخصصة على الموبايل!

HTML5 — أنواع Input الجديدة
<!-- تاريخ --> <input type="date"> <!-- وقت --> <input type="time"> <!-- لون --> <input type="color"> <!-- رقم بحدود --> <input type="number" min="1" max="100"> <!-- شريط تمرير --> <input type="range" min="0" max="100"> <!-- بحث --> <input type="search" placeholder="ابحث هنا..."> <!-- required و placeholder و autofocus --> <input type="email" required placeholder="ادخل إيميلك" autofocus>
